# Bass Catcher

バンドメンバーのベーシストが、楽曲のルート音を耳で拾えず困っていたことをきっかけに開発した、  
**ベースの耳コピと練習を補助するWindowsデスクトップアプリ**です。

音源からベース帯域とルート音候補を解析し、再生位置に合わせて表示します。  
解析結果を正解として確定するのではなく、原曲を聴きながら確認・修正するための手掛かりを増やします。

## 主な機能

- MP3 / WAV / FLAC / M4A / AAC / OGGの読み込み
- 再生・一時停止・シーク
- 5秒戻る・5秒進む
- A-B区間ループ
- テンポ・拍位置・キーの推定
- 拍ごとのルート音候補表示
- ルート音の手動修正
- Demucsによるベース分離
- Basic Pitchを使ったAI併用解析
- 再生テンポのリアルタイム変更
- キー変更を反映した練習用WAVの作成
- 検出ルート音のピアノ通知
- ピアノ通知音量の個別調整
- 通知タイミングの先行補正
- ベースの発音タイミングに追従した通知
- リズム譜調のルート譜PDF出力
- CSV / MusicXML出力
- 解析セッションの保存と再読み込み

## 配布版

Windows向け配布版は、GitHubの **Releases** からダウンロードできます。

1. ZIPファイルを任意の場所へ展開
2. 展開された`BassCatcher`フォルダを開く
3. `BassCatcher.exe`を起動
4. 「音源を読み込む」から楽曲を選択
5. 解析モードを選択
6. 「ルート音を解析」を実行
7. 原曲を聴きながら候補音を確認・修正

配布版を使用するだけであれば、Pythonのインストールは不要です。

`BassCatcher.exe`だけを別の場所へ移動せず、展開したフォルダごと使用してください。

## 基本的な使い方

### 音源を解析する

1. 「音源を読み込む」から楽曲を選択
2. 解析モードを選択
3. 必要に応じてDemucsによるベース分離を有効化
4. 「ルート音を解析」を実行
5. タイムラインと解析一覧で候補音を確認

### 検出音を耳で確認する

「検出音をピアノで鳴らす」をONにすると、解析したルート音を短いピアノ音で確認できます。

- 通知音量は原曲の音量と別に調整できます
- 「通知先行」で端末ごとの再生遅延を補正できます
- ベースの発音タイミングを解析し、音が立ち上がった位置へ通知を合わせます
- 休符や発音がない区間は鳴らしません

通知タイミングが遅く感じる場合は、「通知先行」を少しずつ増やしてください。

### テンポを変更する

再生テンポを0.50〜1.50倍へ変更できます。

再生中に値を変更すると、現在の音源へ即時反映されます。

### キーを変更する

キー変更は原曲へ直接リアルタイム適用せず、練習用WAVへ反映します。

1. キーを-12〜+12半音で指定
2. 「テンポ・キーを反映」を実行
3. 作成された練習音源へ切り替えて確認

練習音源の再生中は、検出ピアノ音も同じ半音数だけ移調されます。

### 解析結果を修正する

誤認識した候補音は、解析一覧から正しい音名または休符へ変更できます。

修正内容はタイムライン、PDF、CSV、MusicXML、解析セッションへ反映されます。

## 解析モード

| 表示 | 内容 |
| --- | --- |
| 高精度解析 | pYINを中心に使用する標準解析 |
| AI併用解析 | Basic PitchとDSP解析を組み合わせる解析 |
| 高速解析 | 処理速度を優先する簡易解析 |

Demucsが利用できる環境では、原曲からベースを分離して解析できます。

## PDF出力

PDFは五線譜や表形式ではなく、バンド練習で追いやすい**リズム譜調のルート進行**として出力します。

- 小節線
- 拍のスラッシュ
- ルート名
- 休符
- 低確信度表示
- 手動修正表示

## セットアップ

```powershell
git clone https://github.com/Ryohei0Otsuka/bass-catcher.git
cd bass-catcher

python -m venv .venv
.\.venv\Scripts\Activate.ps1

.\install.ps1
```

AI機能も導入する場合：

```powershell
.\install.ps1 -AI
```

## 起動

```powershell
.\run.ps1
```

または：

```powershell
python -m app.main
```

## Windows EXEの作成

通常版：

```powershell
.\build_exe.ps1
```

AI機能込み：

```powershell
.\build_exe.ps1 -AI
```

生成先：

```text
dist\BassCatcher\BassCatcher.exe
```

配布時は、`dist\BassCatcher`フォルダ全体をZIPへまとめてください。

## 注意事項

音源のミックス、ベースの音量、歪み、キックとの重なりなどにより、誤認識が発生する場合があります。

解析結果は正解を保証するものではありません。  
演奏前には必ず原曲を再生し、耳で確認してください。

発音タイミング追従は解析時に計算されます。  
機能更新後に以前の解析結果を使う場合は、音源を再解析してください。

1拍内に複数の8分音符や16分音符がある細かなベースフレーズは、完全には再現できません。

読み込んだ音源を外部サーバーへアップロードする機能はありません。

本アプリは現在開発中の試験版です。
