# Bass Catcher

バンドメンバーのベーシストが、楽曲のルート音を耳で拾えず困っていたことをきっかけに開発した、  
**ベースの耳コピ・練習・ルート譜作成を補助するWindowsデスクトップアプリ**です。

音源を読み込み、ベース帯域とルート音候補を時間軸上へ表示します。  
解析結果を正解として押し付けるのではなく、ベーシストが原曲を聴きながら確認・修正するための手掛かりを増やします。

## 開発の背景

Bass Catcherは、バンドで演奏する楽曲の耳コピにおいて、  
ベーシストがルート音を拾えず困っていたことから始まりました。

目指しているのは、ベースラインを完全自動で確定するソフトではありません。

- 原曲のどこでベースが鳴っているか見つける
- 低音域の候補を再生位置に合わせて追う
- 拍ごとのルート音候補を確認する
- 誤認識した箇所だけ人間が修正する
- 練習用のルート譜や音源を作る

という流れを支援し、耳コピにかかる負担を減らすことを目的としています。

> 自動解析で演奏を決めるのではなく、  
> 人間が耳で確かめるための手掛かりを増やすアプリです。

## 解析方式

Bass Catcherは、単純に「最も低く、最も音圧の高い音」をルート音として採用するわけではありません。

通常解析では、以下を組み合わせています。

- 打楽器成分と持続音成分の分離
- ベース帯域の抽出
- pYINによる基音候補推定
- テンポと拍位置の解析
- 拍頭への重み付け
- 音圧と有声音確率
- 楽曲キーとの整合
- 前後の音程移動
- 短い誤検出の補正
- オクターブ誤認の補正

## 解析モード

| 画面表示 | 内容 |
| --- | --- |
| 高精度解析 | pYINを中心に使用する標準解析 |
| AI併用解析 | Basic PitchとDSP解析を組み合わせる解析 |
| 高速解析 | 処理速度を優先する簡易解析 |

Demucsが導入されている場合は、原曲からベースを分離してから解析できます。

AI関連機能が利用できない場合でも、通常の高精度解析と高速解析は使用できます。

## 配布版EXEの使い方

配布されたZIPを任意の場所へ展開し、次のファイルを起動します。

```text
BassCatcher.exe
```

配布版EXEを使用するだけであれば、Pythonのインストールは不要です。

### フォルダ形式の配布版

```text
BassCatcher/
├─ BassCatcher.exe
└─ その他の実行用ファイル
```

`BassCatcher.exe`だけを別の場所へ移動せず、フォルダごと使用してください。

### 単一EXE形式

```text
BassCatcher.exe
```

単一EXE形式は起動時に内部ファイルを一時展開するため、フォルダ形式より起動に時間がかかる場合があります。

## 基本的な使い方

1. **音源を読み込む**を押す
2. MP3などの音源を選択する
3. 解析モードを選択する
4. 必要に応じて**Demucsでベースを分離**を有効にする
5. **ルート音を解析**を押す
6. タイムラインと右側の一覧で候補音を確認する
7. 誤認識した音を選択し、正しい音名または休符へ修正する
8. A-B区間ループやテンポ変更を使って原曲と聴き比べる
9. PDF、CSV、MusicXMLまたは解析セッションを出力する

## 出力形式

### PDF

4拍単位でルート音候補を並べた、練習用のルート譜を出力します。

### CSV

次の情報を出力します。

- 拍番号
- 開始時刻
- 終了時刻
- ルート音
- MIDI番号
- 確信度
- 音圧
- 解析元
- 手動修正状態

### MusicXML

MuseScoreなどへ読み込み、五線譜として追加編集できます。

### Bass Catcher解析セッション

解析結果と手動修正内容をJSON形式で保存し、後から作業を再開できます。

## 精度について

音源分離や自動採譜には限界があります。

特に次のような音源では誤認識が起こりやすくなります。

- キックとベースの低音が強く重なる
- ベースに強い歪みやエフェクトがかかっている
- ギターやシンセがベース帯域まで鳴っている
- スライド、ゴーストノート、ハンマリングが多い
- コードの最低音とベースのルート音が一致しない
- ライブ録音やノイズの多い音源
- ベースの音量が極端に小さい

解析結果は正解を保証するものではありません。  
演奏前には必ず原曲を再生し、耳で確認してください。

## プライバシー

通常の解析処理はローカル環境で実行します。  
読み込んだ音源を外部サーバーへアップロードする機能はありません。

著作権のある音源を扱う場合は、個人利用やバンド練習など、権利上認められた範囲で使用してください。
